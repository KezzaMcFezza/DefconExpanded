Dedcon Software

So if you are new to Defcon you might be wondering...
What the hell is Dedcon?

Dedcon is a powerful dedicated server tool created by bert_the_turtle for hosting DEFCON games, Recording them and the ability to watch them back. All while using the same program!

All of the credit for the functionality and the bug fixes goes towards him, I (KezzaMcFezza) merely collaborated with bert_the_turtle to modernize Dedcon to the best of our ability.

Update frequency will be slow but new features are on the horizon so keep checking this page for any relevant updates if you are a server hoster or just someone who watches recordings.

To learn more about Dedcon and what it does and its history check out bert's website below

Watching Recordings

Windows Users

To watch recordings just click on the recording file and it will launch a LAN server for you to join!

Linux and MacOS

Download the .tar and extract it anywhere it doesnt matter the location,
then to watch recordings open the terminal and CD into the directory where the dedcon binary lives.
You can invoke dedcon to load a recording by typing...

./dedcon -l exampledrecording.dcrec

And the recording should show up in the lan browser! Remember for 8 and 10 player recordings you will need to invoke dedcon with...

./dedcon8p and use the .d8crec extension.
./dedcon10p and use the .d10crec extension.

Installation Guide

Windows Installation

+

Really simple just run the installer and away you go!

If you want to host servers on windows remember to uninstall the old dedcon completely as the new installer comes with updated scripts.

Linux Installation

+

You can download the .tar and extract the folder anywhere, dedcon does not care where it lives.
And after that you should be good to go! as a linux user you will figure it out im sure :)

MacOS Installation

+

Installing Dedcon for MacOS is the same as Linux.
Just download the .tar and place it anywhere that you want to.
