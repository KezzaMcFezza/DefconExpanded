Smurf Checker

The DefconExpanded smurf detection tool helps identify players who may be using multiple names to evade recognition, known as Smurfing.

Using our already existing game data, we use an algorithm/system that can draw conclusions about whether a player is smurfing or not.

How it works:

The system compares two key data points:

  • KeyID's (readable through the netcode)
  • IP Addresses (also readable in the netcode)

The algorithm then parses through all known entries in the database from those two point data points of the player in question.

Privacy & Security:

In the interest of player security and general morality, we do not share any of these data points to you, we only use them to produce a result everyone can read and use.

Something Wrong?

If you think there is a mistake about the result or have concerns about the smurf detection system, please submit a report below. We will investigate any potential issues with the detection algorithm.
