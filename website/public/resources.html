DefconExpanded Builds

Welcome to our collection of modded DEFCON builds! Here you can download these versions of DEFCON that support 8, 10, and 16 player matches.

These Builds are closely based off MINICOM, a community fixed and expanded upon version of DEFCON created by Bert_the_turtle. These builds include the following features:

  • 16 Player limit, instead of the original 6
  • Includes map mods that contain 7-16 player configurations
  • Better modding support
  • A ton of new game options to play around with
  • Empty bomber launch feature

The most recent releases support MacOS and Linux!

Special thanks to bert_the_turtle for helping with the cross-platform development, making it possible for everyone to enjoy these modded builds regardless of their platform.

If you get a virus warning on Windows builds, temporarily disable Windows Defender as these builds aren't digitally signed. Check out our installation guide below if you are struggling.

These modded builds also support Dedcon so you can host your own servers!

Installation Tutorial

Important Information

Windows Users

If you get a virus warning, temporarily disable Windows Defender. These builds aren't digitally signed so they may trigger false positives. Use the installer for easiest setup, or extract manually to your DEFCON directory.

Download Windows Installer


Linux & MacOS Users

Download the appropriate .tar file and extract it anywhere you prefer. The builds are cross-platform compatible. For MacOS ARM users, the Intel builds work through Rosetta translation

Make sure to backup your original DEFCON installation before installing modded builds.

Installation Guide

Windows Installation

+

Method 1 - Installer (Recommended):
Download and run the .exe installer. It will automatically install the modded build and replace your original DEFCON installation.

Method 2 - Manual Installation:
1. Download the .zip file
2. Extract to your DEFCON installation directory (usually in C:/Steam/steamapps/common/DEFCON/)
3. Overwrite existing files when prompted

Note: You may need to temporarily disable Windows Defender to avoid false virus warnings.

Linux Installation

+

1. Download the archive
2. Extract the archive: tar -xf defcon-expanded-linux.tar
3. Copy the files to your DEFCON installation directory
4. Make sure the executable has proper permissions: chmod +x defcon

The Linux builds are cross-distro compatible and should work on most modern distributions. Built on the oldest Ubuntu version github actions would allow us to use.

MacOS Installation

+

1. Download the archive
2. Extract the archive by double-clicking or using Terminal
3. Copy the files to your DEFCON installation directory
4. On first run, you may need to right-click and select "Open" to bypass Gatekeeper

ARM64 Mac users: Intel builds work through Rosetta 2 translation if native ARM64 builds aren't available.
