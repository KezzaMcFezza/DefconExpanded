DEFCON Smurf Checker

How it works

This tool helps you check if a player name has been used by multiple people (smurfing). Enter a player name to see if they have alternate accounts or if someone else has used their name.

Privacy Note: This tool only shows player names - no IP addresses or authentication keys are displayed to protect player privacy.

Results: The tool shows the total number of potential hits and alternate names detected for the searched player.
