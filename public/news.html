New Update Released

A revamped version of the 8 Player Mod that has been built on MINICOM as a base instead of DEFCON, adding MINICOM features to the 8 player build will make the game play better and more stable. To download the newest version see the Resources tab which includes a tutorial on how to install.

If you would like to know more about MINICOM and what it changes in DEFCON, here is the link to the Official Website.

10 Player Build

The 10 Player build has received alot of changes in the past few weeks and is almost ready for its first public test. The territories have been the main focus of these changes due to game balance most notably shrinking the size of the Antarctic reach for South Africa and Australia/Indonesia territories.

If you would like to try the 10 Player build out yourself, the most recent Version 2.10.1 is available in the Experimental tab of the website.
