WARNING:

The 8k variant can sometimes flat out refuse to load on some systems due to how virtual memory space
works in Windows! Linux may have the same issue however your mileage may vary.

SOMETHING TO NOTE:

If you are using a pre 2010 GPU the 4k variant might also flat out refuse to run due to 4k texture
sizes being relatively uncommon in that era.

THE GOOD:

The 2k and 4k variants should run on any computer made within the past 20 years, if your computer cannot
run the 2k variant however...

Then that's a serious skill issue :)
